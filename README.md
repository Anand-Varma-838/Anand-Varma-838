## Featured projects

### 🔍 [agentic-rag-claude](https://github.com/Anand-Varma-838/agentic-rag-claude)
**Agentic RAG system powered by Claude's tool-use API**

A document intelligence system where Claude autonomously decides *when* to retrieve, *what* to search for, and *how* to chain multiple searches to answer complex questions — going beyond naive retrieve-then-generate pipelines.

- Claude tool-use loop with multi-hop reasoning
- Hybrid BM25 + semantic retrieval merged via Reciprocal Rank Fusion
- Cross-encoder reranking for high-precision results
- Live agent trace viewer in Streamlit UI
- `Python` `Anthropic Claude` `ChromaDB` `sentence-transformers` `Streamlit`
